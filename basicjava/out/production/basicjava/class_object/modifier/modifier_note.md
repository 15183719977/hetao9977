## 访问修饰符
### 一、类之间的关系
* 同包子类：a、b位于同一包下，b为a子类
* 不同包子类：a、b位于不同一包下，b为a子类
* 同包类：a、b位于同一包下，无继承关系
* 其他类：a、b位于不同一包下，无继承关系

### 二、修饰符作用范围
|           | 同包子类 | 不同包子类 | 同包类 | 其他类 |
| --------- | -------- | ---------- | ------ | ------ |
| private   | -        | -          | -      | -      |
| package   | 继承     | -          | 访问   | -      |
| protected | 继承     | 继承       | 访问   |        |
| public    | 继承     | 继承       | 访问   | 访问   |
