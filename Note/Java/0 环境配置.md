# 环境配置

## 环境变量

#### 内容

- `JAVA_HOME`: C:\Program Files\Java\jdk-11.0.4
- `PATH`: %JAVA_HOME_11%\bin

#### 注意

- 多版本共存采用压缩包，不要使用 exe 安装
- IDEA 只认 **JAVA_HOME**，不认 **JAVA_HOME_11** 等

## 命令提示符

#### cd 

- `cd 文件夹路径`： 打开指定文件夹
- `cd ..`：返回上一级
- `cd \`：返回根目录

#### dir

- 查看文件文件夹信息

#### cls

- 清空 CMD 屏幕

#### exit

- 退出
