#### 面向对象

1. 创建对象

   - ```java
     public class Hero{//创建类
     	String name;//类的属性
     	
     	public static void main(String[] args){
     		Hero xu = new Hero();//创建对象
     		xu.name = "xuxu";
     	}
     }
     ```



2. 面向对象方法
   
   - ```java
     public class Hero{//创建类
     	String name;//类的属性
     	
     	void shuai(){
     		System.out.println("帅气");
     	}//类的方法
     	public static void main(String[] args){
     		Hero xu = new Hero();//创建对象
     		xu.name = "xuxu";
     		xu.shuai();//调用对象的方法
     	}
     }
     ```
