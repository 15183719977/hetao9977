#### HelloWorld 案例

1. 程序的代码编译

   - 开发步骤：编写源程序、编译（javac.exe）为字节码文件、运行(java.exe)字节码文件

   - 编写源程序：见“.java”文件

   - 编译：

     ```java
     javac Helloworld.java
     ```

   - 运行：

     ```java
     java Helloworld        //无后缀
     ```

     

2. 注释

   + 单行注释

     ```java
     public class HelloWorld{
     
     }    //这是单行注释
     ```

     

   + 多行注释
     
     ```java
     /*
     这是多行
     注释
     */
   public class HelloWorld{
     
     }    
     ```
