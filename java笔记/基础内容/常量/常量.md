# 常量

|    类型    |                  含义                  |      例子      |
| :--------: | :------------------------------------: | :------------: |
|  整数常量  |                所有整数                |  1，100，-423  |
|  小数常量  |                所有小数                | 0.0, 0.23,-0.1 |
|  字符常量  |   单引号引起来的一个字符，必须有内容   | 'a', '我'，‘ ’ |
| 字符串常量 | 双引号引起来的，可以多个字符，也可以空 | “a”, “abc”, “” |
|  布尔常量  |               只有两种值               |  true , false  |
|   空常量   |               只有一个空               |      null      |
