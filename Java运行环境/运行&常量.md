# 常量
- 字符串常量：System.out.println("");`可以为空`
- 整数常量：System.out.println(300);
- 浮点数常量：System.out.println("3.14");
- 字符常量：System.out.println('A');`不能为空`
- 布尔常量：System.out.println(true);
- 空常量：null：不能直接打印输出
