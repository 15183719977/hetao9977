# 继承与多态

- 面向对象的**三大特征**：封装性、继承性、多态性。

## 继承

- **继承是多态的前提**，如果没有继承，就没有多态。
- 继承主要解决的问题就是：共性抽取。
- 继承关系当中的特点：
  1. 子类可以拥有父类的“内容”；
  2. 子类还可以拥有自己专有的内容。

- **继承的格式**：

  - 定义父类的格式：（一个普通的类定义）

  ```
  public class 父类名称{
  	//……
  }
  ```

  - 定义一个子类的格式：

  ```
  public class 子类名称 extends 父类名称{
  	//……
  }
  ```
